# 输入方案

输入方案管理允许用户添加、切换和管理不同的输入方案（如五笔86、五笔拼音混输、拼音等）。

## 方案列表

在「设置」→「方案与词库」→「输入方案」中，可以看到所有可用的输入方案：

- **已启用** — 当前已开启的方案，显示为已启用列表
- **未启用** — 存在于 `rime/shared/` 目录中但尚未启用的方案

每个方案使用右侧的 Switch 开关来启用或停用。点击删除按钮可以从设备上移除该方案及其词典文件。

> 提示：修改启用状态后，点击底部的「部署方案」按钮使更改生效。

## 导入方案

支持多种方式导入新的输入方案：

### 从文件选择

1. 点击右上角菜单 →「从文件选择」
2. 选择 `.zip` 压缩包（包含 `.schema.yaml` 和 `.dict.yaml`）
3. 系统会自动解压并提取方案文件到 `rime/shared/`
4. 导入后方案会出现在「未启用」列表中，需要手动启用并部署

### 无线导入

1. 点击右上角菜单 →「浏览器导入」
2. 在弹出的页面中查看二维码和上传地址（如 `http://192.168.1.100:5000`）
3. 在同一 Wi-Fi 下的电脑浏览器中打开该地址
4. 拖拽或选择 `.zip` 或 `.tar.gz` 压缩包上传
5. 上传完成后手机端会自动更新方案列表

### 网络导入

1. 点击右上角菜单 →「网络导入」
2. 在弹出的对话框中输入压缩包的下载 URL
3. 支持 `.zip` 和 `.tar.gz` 两种格式
4. 点击确定后自动下载、解压并更新方案目录

## 切换方案

在键盘上：

1. 点击键盘上的 Logo 按钮打开功能菜单
2. 菜单中显示已启用且已部署的方案名称
3. 点击即可切换到对应方案

在设置中：

1. 进入「输入方案」设置页面
2. 在「已启用」列表中使用 Switch 开启或关闭方案
3. 点击底部「部署方案」按钮使更改生效

## 部署方案

每次添加、启用或删除方案后，都需要执行部署操作才能生效：

1. 在「输入方案」页面底部点击「部署方案」
2. 等待部署完成（通常需要 10-30 秒）
3. 部署完成后新的方案即可使用

## 删除方案

1. 在「未启用」列表中找到要删除的方案
2. 点击删除图标
3. 确认删除后，该方案的 `.schema.yaml` 和 `.dict.yaml` 文件会从设备上移除
4. 如果方案已启用，需要先在已启用列表中关闭

## 注意事项

- 添加方案后，需要手动在「已启用」列表中开启，然后部署才能生效
- 导入的压缩包应包含 `.schema.yaml` 和对应的 `.dict.yaml` 文件
- 部署过程需要几秒到几十秒，请耐心等待
- 切换方案后必须执行部署才能生效
