# 部署方案

部署方案功能用于编译 Rime 输入方案，让修改后的词库和配置生效。不过，由于暂时未开放自定义输入方案，因此此功能暂时还无效。

## 功能特性

- **编译方案** - 编译用户方案配置文件（如 `.schema.yaml`, `.dict.yaml`）
- **生成文件** - 生成编译后的文件（如 `.prism` 和 `.table` 文件）
- **应用更改** - 修改词库、方案配置后需要部署才能生效

## 使用方法

1. 点击键盘候选栏左侧的 Logo 图标打开功能菜单
2. 选择"部署方案"
3. 等待几秒钟，屏幕会显示"方案部署中..."
4. 部署完成后会显示"方案部署完成"

## 需要部署的场景

- 新增或修改用户词库（`.dict.yaml`）
- 修改输入方案配置（`.schema.yaml`）
- 添加或更新表情插件
- 切换输入方案后（部分情况）

## 注意事项

- 部署过程需要几秒钟，请耐心等待
- 部署期间键盘会自动隐藏
- 如果部署失败，请检查方案配置文件格式是否正确
